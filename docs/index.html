* Definitions

Online
digitized materials, webpages, finding aids, e-resources, other online resources
Not online
links to catalog other descriptive records (other than finding aids) but objects themselves aren't online.
GMD.MAR
A grouping for processing purposes, which holds all online materials from LOC's Geography and Map Division that are not otherwise held in another processing group. In API query results, can be found in "group" field.All GMD.MAR objects are online.
